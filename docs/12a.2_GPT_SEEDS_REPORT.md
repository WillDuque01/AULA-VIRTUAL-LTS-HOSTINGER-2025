## Cómo ejecutar las semillas
1. **Opcional (limpiar datos previos)**  
   ```bash
   php artisan migrate:fresh
   ```
2. **Sembrar los perfiles de auditoría**  
   ```bash
   php artisan db:seed --class=AuditorProfilesSeeder
   ```
   (También puedes usar `php artisan db:seed` sin parámetros porque `DatabaseSeeder` ya incluye este seeder).
